p5 simple site editors.

Kleine, herbruikbare p5.js-editors — elk een op zichzelf staand building block. Van het simpelste live-editortje tot een PHP-editor die echte bestanden bewerkt.

Basis

client-side

De simpelste vorm: één code-veld met een live preview, Run en Reset.

p5-sketch-editor/

Partial

client-side

Toont enkel een gemarkeerd blok (//show … //end); de rest van de sketch draait verborgen. Voor gerichte oefeningen.

p5-sketch-editor_partial/

Multi-file

client-side

Drie bewerkbare bestanden (HTML / CSS / JS) via tabs; de preview combineert ze.

p5_sketch_editor_ext/

All-feature

PHP

Bewerkt echte bestanden op schijf: multi-file tabs, opslaan, live preview en Sketch/Bestand-menu's (nieuw, dupliceren, hernoemen, verwijderen).

⚠ Werkt niet op GitHub Pages — vereist een lokale PHP-server (XAMPP/Apache). Deze kaart linkt naar de broncode.

p5_sketch_editor_php/
Broncode op GitHub. Elke editor heeft een eigen README met details.
